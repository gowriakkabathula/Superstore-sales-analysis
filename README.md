# Superstore-sales-analysis
Excel-based Sales Dashboard project using pivot tables, KPIs, and data analysis to generate business insights.
## Excel Sales Dashboard Project
This project is an Excel-based Sales Dashboard created to analyze business performance.
The dashboard provides insights into sales, profit, orders and top 10 products in a simple and visual way.
## Tools Used
Microsoft EXcel
Pivot Tables
Pivot Charts
KPI Metrics
Data Cleaning
## Key Metrics(KPIs)
1. Total Sales
2. Total Profit
3. Total Orders
4. Profit Margin
## Key Insights
1. The business generated consistent sales across different categories.
2. Profit Margin is around 30%, indicating moderate profitability.
3. Some categories and products perform better than others in terms of profit.
4. Order Volume is high, showing good customer demand.
## Dashboard Features
1. Interactive filters for better analysis
2. Visual Charts for sales and profit trends
3. KPI cards for quick overview
4. Clean and simple design for easy understanding
